# 📥 Руководство по участию в разработке Downloader

Спасибо, что решили помочь в развитии проекта! Этот документ содержит основные рекомендации по работе с кодом.

## Как помочь проекту

1.  **Отчеты об ошибках**: Если вы нашли баг, создайте Issue с подробным описанием проблемы и шагами для её воспроизведения.
2.  **Предложения функций**: Если у вас есть идеи по улучшению, открывайте обсуждение в Issue.
3.  **Pull Requests**:
    *   Сделайте Fork репозитория.
    *   Создайте новую ветку для вашей фичи или исправления.
    *   Создавайте коммиты с понятными описаниями.
    *   Отправьте Pull Request в ветку `main`.

## Настройка окружения

1.  Установите Node.js (v18 или выше).
2.  Запустите скрипт настройки:
    *   На Windows: `setup.bat`
    *   На Mac/Linux: `./setup.sh`
3.  Запустите проект: `npm start`

## Стандарты кода

- Используйте понятные имена переменных и функций на английском языке (в коде).
- Все строки интерфейса (UI) должны быть на **русском языке**.
- Сохраняйте темную тему оформления (Bento/Discord style).

## Лицензия

Участвуя в разработке, вы соглашаетесь, что ваш код будет распространяться под лицензией MIT.
