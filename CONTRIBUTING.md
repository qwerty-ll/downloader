# Contributing to Downloader Pro

Thank you for your interest in contributing to Downloader Pro! Here is a set of guidelines for contributing to this project.

## How to Contribute

1. **Bug Reports**: If you find a bug, please create an issue with a detailed description of the problem and steps to reproduce.
2. **Feature Requests**: If you have an idea for a new feature, feel free to open an issue for discussion.
3. **Pull Requests**:
   - Fork the repository.
   - Create a new branch for your feature or fix.
   - Commit your changes with clear and descriptive messages.
   - Submit a pull request to the `main` branch.

## Development Setup

1. Install dependencies: `npm install`
2. Run in development mode: `npm start`
3. Build for distribution: `npm run dist`

## Coding Standards

- Use clear and descriptive variable/function names.
- Ensure all UI changes follow the existing Tailwind CSS dark mode theme.
- Test your changes on both macOS and Windows if possible.
