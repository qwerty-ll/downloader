# 📥 Downloader Pro

**Downloader Pro** — это профессиональное десктопное приложение для сверхбыстрой загрузки видео и потоков. Минимум настроек, максимум скорости. Проект полностью переведен на русский язык и оптимизирован под удобство обычного пользователя.

![Downloader Pro UI](/Users/smirnovmakar/.gemini/antigravity/brain/5631a614-019d-4409-a345-b3fb373cea19/downloader_pro_ui_mockup_1776449385694.png)

## ✨ Основные преимущества

-   **Zero-Config**: Вам не нужно ничего устанавливать отдельно. Скрипт настройки сам скачает портативные версии `yt-dlp`, `ffmpeg` и `aria2c`.
-   **Turbo Mode**: Использование мультипоточного скачивания через Aria2c для полной загрузки вашего интернет-канала.
-   **Полная русификация**: Весь интерфейс, уведомления и логи переведены на русский язык.
-   **Обход ограничений**: Встроенная поддержка Cookies позволяет скачивать приватные видео, просто выбрав ваш браузер в списке.
-   **Умный трекинг**: Реальное отображение прогресса, скорости загрузки и логов ошибок прямо в окне программы.

## 🚀 Быстрый старт (для новичка)

Мы максимально упростили процесс запуска. Вам нужно только:

1.  Установить [Node.js](https://nodejs.org/) (версия 18+).
2.  Скачать этот проект и запустить **один файл**:
    -   На Windows: запустите **`setup.bat`**
    -   На macOS / Linux: запустите **`setup.sh`** в терминале.

**Скрипт сам:**
- Установит все компоненты.
- Скачает актуальные версии загрузчиков (`yt-dlp`, `ffmpeg`, `aria2c`) в папку проекта.
- Проверит готовность системы.

После этого просто введите команду для запуска:
```bash
npm start
```

## 🛠 Ручная настройка (для опытных)

Если вы хотите использовать свои системные версии утилит:
1.  Установите зависимости: `npm install`
2.  Проверьте окружение: `npm run setup`
3.  Запуск: `npm start`

## 📂 Структура проекта

-   `bin/`: Папка для портативных утилит (создается автоматически).
-   `main.js`: Главный процесс (Back-end).
-   `renderer.js`: Логика интерфейса (Front-end).
-   `scripts/download-tools.js`: Скрипт авто-загрузки зависимостей.
-   `index.html`: Интерфейс приложения.
