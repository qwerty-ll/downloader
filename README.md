# 📥 Downloader Pro

**Downloader Pro** — это мощное и стильное десктопное приложение для максимально быстрой загрузки видео с YouTube и тысяч других платформ. Построено на базе Electron и использует всю мощь `yt-dlp`, `ffmpeg` и `aria2c`.

![Downloader Pro UI](/Users/smirnovmakar/.gemini/antigravity/brain/5631a614-019d-4409-a345-b3fb373cea19/downloader_pro_ui_mockup_1776449385694.png)

## 🚀 Особенности

-   **Максимальная скорость**: Поддержка многопоточной загрузки через `aria2c` (Turbo Mode).
-   **Премиальный дизайн**: Современный Dark Mode в стиле Discord/VS Code на базе Tailwind CSS.
-   **Универсальность**: Работает на Windows, macOS и Linux.
-   **Обход ограничений**: Встроенная поддержка подгрузки куки из ваших браузеров (Chrome, Firefox, Safari и др.) для скачивания приватных или ограниченных видео.
-   **Контроль в реальном времени**: Живое отображение процентов, скорости загрузки и статуса системных утилит.

## 🛠 Требования (Dependencies)

Для работы приложения в системе должны быть установлены следующие утилиты:

1.  **yt-dlp**: Основной движок загрузки.
2.  **FFmpeg**: Для склейки видео и аудио дорожек высокого качества.
3.  **Aria2c**: Для активации Turbo-режима (многопоточность).

### Как установить зависимости:

#### macOS (через Homebrew):
```bash
brew install yt-dlp ffmpeg aria2
```

#### Windows (через Scoop):
```bash
scoop install yt-dlp ffmpeg aria2
```
*Или скачайте .exe файлы и добавьте пути к ним в системную переменную PATH.*

## 📦 Быстрая установка (One-Click Setup)

Чтобы максимально быстро запустить проект со всеми зависимостями, используйте автоматические скрипты:

### Для Windows:
Просто запустите файл **`setup.bat`** двойным кликом.

### Для macOS / Linux:
Запустите файл **`setup.sh`** в терминале:
```bash
./setup.sh
```

Эти скрипты сами:
1.  Установят все библиотеки через `npm install`.
2.  Проверят наличие системных утилит (`yt-dlp`, `ffmpeg`, `aria2c`).
3.  Подскажут, если чего-то не хватает.

## 📖 Как пользоваться

1.  Вставьте ссылку на видео в поле ввода.
2.  Выберите папку для сохранения (кнопка "Browse").
3.  При необходимости выберите браузер для импорта куки (если видео требует авторизации).
4.  Включите **Turbo Mode**, если хотите использовать всю мощь вашего интернет-канала.
5.  Нажмите **START DOWNLOAD** и наслаждайтесь скоростью!

## 📂 Структура проекта

-   `main.js`: Логика главного процесса Electron (управление окнами, IPC, spawn процессов).
*   `preload.js`: Безопасная прослойка между системными функциями и UI.
*   `index.html`: Интерфейс приложения (Tailwind CSS).
*   `renderer.js`: Логика обработки действий пользователя и обновление прогресса.
*   `package.json`: Конфигурация сборки и зависимости.
